# AIを使った動画分析 体験入学コース

カメラ映像をAIでリアルタイム解析するWebアプリを、ステップごとに作っていくコースです。  
コメントアウトされたコードを外したり、空文字列を埋めたりしながら、少しずつ機能を追加していきます。

---

## 実行方法（共通）

ターミナルで以下のコマンドを入力します。`ファイル名` の部分をそれぞれのステップに合わせて変えてください。

```bash
streamlit run ファイル名.py
```

ブラウザが自動で開きます。開かない場合は表示されたURL（`http://localhost:8501`）を開いてください。  
停止するには、ターミナルで `Ctrl + C` を押します。

## ステップ一覧

### Step 1 ── はじめてのWebアプリ `step1.py`

**テーマ：** Streamlit を使って、名前を入力するとあいさつしてくれるページを作る。

**やること：**
1. コメントアウトされている `st.title(...)` と `st.write(...)` の `#` を外して、タイトルと説明文を表示する
2. `name = st.text_input(...)` の `#` を外して、名前入力欄を作る
3. `submitted = False` を、`submitted = st.form_submit_button(...)` に置き換えてボタンを作る
4. `st.selectbox` の空文字列 `""` を「おはようございます！」「さようなら！」などに書き換えて、選択肢を追加する

**確認ポイント：** ボタンを押すと、入力した名前と選んだあいさつが緑のメッセージで表示される。

```bash
streamlit run step1.py
```

---

### Step 2 ── カメラを表示しよう `step2.py`

**テーマ：** PCのカメラ映像をブラウザにリアルタイムで表示する。

**やること：**
1. まずそのまま実行してカメラ映像が表示されることを確認する
2. `cv2.cvtColor(...)` の `#` を外して、映像をモノクロにしてみる
3. `cv2.flip(frame, 0)` の `#` を外して、映像を上下反転させてみる

**確認ポイント：** 「カメラ起動」「カメラ停止」ボタンで映像が切り替わる。

```bash
streamlit run step2.py
```

---

### Step 3 ── 顔を検出しよう `step3.py`

**テーマ：** AI（MediaPipe）で顔を検出し、顔の位置に絵文字を表示する。

**やること：**
1. そのまま実行して「顔の検出」ボタンを押し、カメラに顔を向ける
2. `mp_drawing.draw_detection(...)` の `#` を外して、顔に四角い枠を描く
3. `face_emoji = ""` の空文字列を好きな絵文字（例：`"😊"` や `"🤖"`）に変えて、顔の上に絵文字を表示する

**確認ポイント：** 顔を検出すると、顔の位置に指定した絵文字が表示される。

```bash
streamlit run step3.py
```

---

### Step 4 ── 手を検出して指に名前をつけよう `step4.py`

**テーマ：** AI（MediaPipe）で手の骨格を検出し、各指先にラベルを表示する。

**やること：**
1. そのまま実行して「手の検出」ボタンを押し、カメラに手をかざす
2. `mp_drawing.draw_landmarks(...)` の `#` を外して、手の骨格を描く
3. `finger_names = ['', '', '', '', '']` の空文字列を指の名前（例：`'親指'`, `'人差し指'`, `'中指'`, `'薬指'`, `'小指'`）に変える
4. `for name, tip_idx in zip(...)` ブロックの `#` を外して、各指先に名前を表示する
5. `processed_image[:] = 0` の `#` を外すと背景が黒くなって、骨格だけ表示されるのを試してみる

**確認ポイント：** 手の骨格が描かれ、各指先に名前のラベルが表示される。

```bash
streamlit run step4.py
```

---

### Step 5 ── ストレッチ回数カウンター `step5.py`

**テーマ：** AI（MediaPipe）で体の姿勢を検出し、腕のストレッチ回数を自動で数える。

**やること：**
1. そのまま実行して「ストレッチ」ボタンを押し、腕を上げ下げしてみる
2. カウントが正しく増えるか確認する
3. `processors/pose_counter.py` を開いて、中身を読んでみよう（どうやって姿勢を判定しているか）

**確認ポイント：** 腕を上げて水平にするたびに、画面のカウンターが増える。

```bash
streamlit run step5.py
```

---

### Step 6 ── バーチャル・サングラスをかけよう `step6.py`

**テーマ：** AI（MediaPipe FaceMesh）で顔の細かいパーツを検出し、サングラス画像を顔に合成する。

**やること（3ステップ）：**

1. **ステップ1：顔のメッシュを表示する**  
   `mp_drawing.draw_landmarks(...)` ブロック（5行）の `#` を外して、顔に緑のメッシュを表示する

2. **ステップ2：目の位置を確認する**  
   `cv2.circle(...)` 2行の `#` を外して、両目の位置に赤い丸を表示する

3. **ステップ3：サングラスを合成する**  
   `processed_image[...] = combined` の `#` を外して、サングラスを顔に合成する

**確認ポイント：** カメラに顔を向けると、顔の動きに追従してサングラスが表示される。

```bash
streamlit run step6.py
```

## 応用コース `advanced/`

Step 1〜6 が終わったら、応用コースに挑戦しよう！

---

### エアペイント `advanced/air_paint.py`

**テーマ：** 手の動きだけで空中に絵を描く。親指と人差し指を「つまむ」と描画ON、離すと描画OFF。

**操作方法：**
- サイドバーの「🎮 開始」でカメラを起動
- 親指と人差し指を近づける（ピンチ）と描画モードになる
- 色・太さ・透明度はサイドバーで変更できる
- 「🧹 クリア」でリセット、「↩️ アンドゥ」で一つ戻る

```bash
streamlit run advanced/air_paint.py
```

---

### じゃんけんAI `advanced/janken_ai_main.py`

**テーマ：** カメラに向かって手でグー・チョキ・パーを出すと、AIと対戦できるじゃんけんゲーム。  
ただし、**`advanced/janken_logic.py` の中身が未完成**です。ゲームを動かすには、まずロジックを実装しよう！

**`advanced/janken_logic.py` でやること：**

1. **`classify_hand_gesture()` 関数** ── 伸びている指の本数で手を判定する
   - `count == 0` のとき → `"GU"`（グー：指が全部閉じている）を返す
   - `count == 2` のとき → `"CHOKI"`（チョキ：2本指）を返す  
   - `count >= 4` のとき → `"PA"`（パー：指が全部開いている）を返す

2. **`decide_winner()` 関数** ── プレイヤーとAIの手を比べて勝敗を決める
   - 空文字列 `""` になっている部分に、正しいグー・チョキ・パーの文字（`"GU"`, `"CHOKI"`, `"PA"`）を入れる
   - 戻り値の `""` に `"WIN"` や `"LOSE"` を入れる

```bash
streamlit run advanced/janken_ai_main.py
```

**確認ポイント：** 4秒のカウントダウン後に判定が行われ、スコアが更新される。

## ファイル構成

```
videoanalyze/
├── step1.py              # Step 1: はじめてのWebアプリ
├── step2.py              # Step 2: カメラ表示
├── step3.py              # Step 3: 顔の検出
├── step4.py              # Step 4: 手の検出・指ラベル
├── step5.py              # Step 5: ストレッチカウンター
├── step6.py              # Step 6: バーチャルサングラス
├── advanced/
│   ├── air_paint.py      # 応用: エアペイント
│   ├── janken_ai_main.py # 応用: じゃんけんAI（メイン）
│   └── janken_logic.py   # 応用: じゃんけん判定ロジック（要実装）
├── processors/
│   ├── emoji_drawer.py   # Step 3 で使う絵文字描画モジュール
│   ├── pose_counter.py   # Step 5 で使う姿勢カウントモジュール
│   └── cartoon_styler.py # 追加機能用モジュール
├── glasses/              # サングラス画像（Step 6 で使用）
├── documents/            # 授業スライド・資料
└── requirements.txt      # 必要なライブラリ一覧
```
